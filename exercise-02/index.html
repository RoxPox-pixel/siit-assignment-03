Part 2 - functions, rock paper, scissors game

The exercise should continue the implementation of the rock - paper - scissors game:



Set the validInput constant / variable, depending on the actual user input.

Valid inputs are: “rock”, “paper”, “scissors”. As a bonus, consider valid inputs any form of case insensitive characters for the specified options (e.g. "Rock", "ROCK", "rOcK", etc)

The function should return one of the following strings: "User wins!" / "Computer wins!" / "It's a tie!"
