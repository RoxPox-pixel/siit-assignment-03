Primul exercitiu


Part 1 - functions, array algorithms

In your local repository, create a folder called exercise-01. Inside the folder, add index.html & main.js files. The exercise should implement the following problem:

In the main.js file, implement a function that receives as parameter an array of numbers and returns the maximum value in the array.

Restrictions:

You are not allowed to use Math.max() native method to find the maximum value

You are not allowed to implement the code for reading the values in the array.

The function will be tested by passing as argument a JS array like [2, -1, 10, 6, -3, 8, 5]

Example:

If the function name is maxInArray, calling maxInArray([2, -1, 10, 6, -3, 8, 5]) should return 10.

After implementing the solution, commit the changes and push them to the public repository. Use “exercise 1” as the name for the commit.
